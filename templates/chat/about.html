Acerca de ChatLLM

¿Qué es ChatLLM?

ChatLLM es una aplicación web de chat inteligente construida con Django y HTMX que permite interactuar con múltiples modelos de lenguaje de gran escala (LLM) de forma gratuita a través de la API de NVIDIA Build.

Funcionalidades

  • Autenticación: Registro e inicio de sesión de usuarios.
  • Gestión de conversaciones: Crear, renombrar, archivar, fijar y eliminar conversaciones.
  • Búsqueda: Buscar conversaciones por título o contenido de mensajes.
  • Streaming en tiempo real: Las respuestas del modelo se muestran token a token usando SSE/HTMX.
  • Selector de modelo: Cada conversación puede usar un modelo diferente.
  • Contexto adicional: Añade fragmentos de texto como contexto al prompt.
  • Valoración de respuestas: Marca respuestas como útiles o no útiles.
  • Exportación: Exporta conversaciones en formato Markdown o JSON.
  • Panel de métricas: Estadísticas de uso para administradores.

Tecnologías

  • Django 6.0 + SQLite
  • HTMX 1.9 para interacción dinámica
  • Bootstrap 5.3
  • NVIDIA Build API (modelos gratuitos)
  • Server-Sent Events (SSE) para streaming
